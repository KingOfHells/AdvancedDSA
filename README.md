# AdvancedDSA

A curated collection of **Advanced DSA and problem-solving solutions** based on important LeetCode questions. This repository is organized topic-wise so it is easy to revise concepts like recursion, dynamic programming, stacks, queues, strings, sliding window, and mathematics.

## Problems Covered

| No. | Problem | Topic | What You Learn |
|---|---|---|---|
| 1 | [Factorial Trailing Zeroes](https://leetcode.com/problems/factorial-trailing-zeroes/) | Recursion / Math | Mathematical decomposition and factorial insight |
| 2 | [Fibonacci Number](https://leetcode.com/problems/fibonacci-number/) | Recursion / DP | Difference between recursion and dynamic programming |
| 3 | [Valid Parentheses](https://leetcode.com/problems/valid-parentheses/) | Stack | Stack implementation and LIFO behavior |
| 4 | [Implement Queue using Stacks](https://leetcode.com/problems/implement-queue-using-stacks/) | Queue / Stack | Internal difference between queue and stack |
| 5 | [Valid Palindrome](https://leetcode.com/problems/valid-palindrome/) | Strings / Two Pointers | String cleanup and two-pointer approach |
| 6 | [Maximum Subarray](https://leetcode.com/problems/maximum-subarray/) | Dynamic Programming | Kadane's Algorithm and DP thinking |
| 7 | [Reverse String](https://leetcode.com/problems/reverse-string/) | Strings / Two Pointers | In-place operations and stack usage |
| 8 | [Coin Change](https://leetcode.com/problems/coin-change/) | Dynamic Programming | DP optimization for minimum coins |
| 9 | [Find Greatest Common Divisor of Array](https://leetcode.com/problems/find-greatest-common-divisor-of-array/) | Math / Recursion | Euclidean algorithm and recursion |
| 10 | [Longest Substring Without Repeating Characters](https://leetcode.com/problems/longest-substring-without-repeating-characters/) | Sliding Window / Hashing | Window optimization and hash map usage |
